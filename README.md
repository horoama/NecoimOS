# NecoimOS
30日OS本を参考にゴリゴリOSっぽいものを作ってみる．
基本的に本の30日目を目指して実装．寄り道して違う機能を実装する可能性もあり．

##開発環境
* Linux上で開発
* アセンブリはgas・intel記法で記述
* gccでコンパイル
* 仮想マシンはQEMU
